Majorsilence Reporting — Report Definition Language (RDL) 2025

December 2025

Description

This schema describes the structure of the Report Definition Language (RDL) used by Majorsilence Reporting, an XML schema for representing reports which includes query, calculation and layout metadata.

See http://reporting.majorsilence.com/schemas/reporting/2025/12/reportdefinition for the RDL 2025 variant. Current version: ReportDefinition.xsd
